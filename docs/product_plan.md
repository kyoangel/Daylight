# Daylight 產品規劃（草案）

更新日期：2026-02-03

## 專案定位
- 目標對象：較憂鬱、需要正向支持的使用者；長期在家中獨處的身心障礙者
- 平台：Web / Android / iOS
- 語言與市場：台灣繁中為主，兼顧英語使用者
- 產品定位：偏向自助式療癒工具，未來可加入陪伴式與正念引導

---

## 目前 App 狀態（程式碼觀察）
- Flutter 多平台架構完成：已存在 `onboarding / daily / companion / diary / sos / profile` 路由與底部導航
- 多數頁面仍為 placeholder（清單或簡單文字）
- 已有主題色與個人檔案（暱稱、頭像）儲存（本機 SharedPreferences）
- 有基本 ViewModel 架構，但功能邏輯尚未實作（Daily/Onboarding/SOS/Diary）
- 尚未建立資料儲存策略（雲端/離線同步）、內容系統、心理支持工具邏輯

---

## 產品架構與核心模組
### IA（資訊架構）
- 日常：每日節奏、情緒追蹤、小任務
- 陪伴：互動對話式支持（非醫療替代）
- 日記：情緒紀錄、回顧、引導式寫作
- SOS：求助支援、危機工具、緊急聯絡
- 我：個人化、設定、付費、內容管理

### 個人化設計
- 資料收集（低侵入）：
  - 情緒基準小問卷（3-5 題）
  - 觸發源選擇（孤獨、身體不適、人際壓力等）
  - 喜好互動方式（文字/語音/圖片/任務）
  - 作息與使用時段
- 個人化功能：
  - 陪伴語氣選擇（溫柔/鼓勵/陪聊）
  - 提醒時段（早/中/晚/睡前）
  - 主題色、低刺激/高對比模式
  - 內容推薦（依情緒狀態）

### 心理支持功能模組
- 微介入工具箱（Micro-interventions）
  - 3 分鐘安定練習（呼吸、感官回錨）
  - 情緒命名卡（幫使用者說出感覺）
  - CBT 風格提問（我現在的想法是？）
  - 每日肯定語
- 情緒日記 + 週回顧
  - 今日心情滑桿 + 單句紀錄
  - 週報視覺化（熱度/觸發統計）
  - 引導式寫作模板（感謝/放下/希望）
- 陪伴助手（對話功能）
  - 固定安全話術 + 引導式流程
  - 10 分鐘陪伴模式
  - 內建提示引導到 SOS
- SOS 模組
  - 一鍵求助 + 預先訊息
  - 自我安撫流程（5 分鐘）
  - 清楚標示區域化危機資源

## 付費解鎖策略（Freemium）
- 免費：基本日常、簡易日記、基本陪伴
- 付費：
  - 深度情緒回顧與週報
  - 進階工具包（睡眠、焦慮、低落）
  - 個人化內容推薦與語音
  - 新功能優先體驗
- 收費方式：
  - 月/年訂閱
  - 一次性解鎖（工具包/主題包）
  - 支援方案（Supporter plan）

## 黏著度設計
- 每日例行（短且可完成）：一句話 + 心情打卡
- 連續使用回饋：連續 3 天給溫柔徽章
- 週末儀式：自動生成「這週的我」卡片
- 小任務：像「今天幫自己泡杯水」
- 回顧價值：使用者可看到成長軌跡

## 變現方向（非訂閱）
- 品牌合作：心理健康內容包
- 企業/機構方案：照護機構或 NGO 合作
- 專業課程包：與心理師合作
- VIP 陪伴：真人社群或課程

## MVP 優先順序（建議）
1. Onboarding：個人化基礎資料收集
2. Daily：每日心情打卡 + 小任務
3. Diary：情緒日記 + 週回顧
4. SOS：一鍵求助 + 安定流程
5. Profile：主題 + 提醒時間

## 具體介面草圖分鏡（文字版）
以下以「主要流程 + 核心畫面」的分鏡描述，便於後續轉成設計稿。

### A. Onboarding（首次引導）
1. 歡迎頁
- 標題：一日之光
- 副標：你不是一個人，我們會陪你走一小步
- CTA：開始

2. 基本設定
- 暱稱
- 使用語言：繁中 / English
- 使用時段：早 / 中 / 晚 / 睡前
- CTA：下一步

3. 情緒偏好
- 最近常見心情（多選）
- 觸發源（多選）
- 喜好互動方式（文字/語音/圖片/小任務）
- CTA：完成

4. 安全提示
- 文字：這不是醫療服務，若你需要緊急協助，請使用 SOS
- CTA：進入首頁

### B. 日常（Daily）
1. Daily 首頁
- 今日心情滑桿（0-10）
- 今日一件小任務（可替換）
- 今日一句話（肯定語）
- CTA：完成打卡

2. 小任務完成
- 動畫回饋（簡短、溫柔）
- 「今天的我值得肯定」卡片

3. 晚安反思
- 1 句引導提問（例如：今天哪一刻你感覺比較輕鬆？）
- 文字輸入欄

### C. 陪伴（Companion）
1. 入口頁
- 選擇模式：
  - 我要陪伴 10 分鐘
  - 我想安靜整理一下
  - 我只想被傾聽

2. 對話頁
- 對話泡泡 + 安全提示
- 快捷回覆（我想聊更多 / 我想休息）
- SOS 快捷入口

### D. 日記（Diary）
1. 日記首頁
- 今日心情選擇 + 文字日記
- 可選模板：感謝 / 放下 / 希望

2. 週回顧
- 情緒趨勢圖
- 常見觸發統計
- 一句週結語（自動生成）

### E. SOS（求助）
1. SOS 首頁
- 一鍵求助（送出預設訊息）
- 自我安撫流程（5 分鐘）
- 危機資源（區域化）

2. 自我安撫流程
- 呼吸引導（動畫）
- 身體掃描（文字/語音）
- 回錨提示（現在你在這裡）

### F. 我（Profile）
- 暱稱、頭像
- 主題色與模式（低刺激/高對比）
- 提醒設定
- 付費升級入口
- 語言切換

## Feature Backlog（分階段）

### P0（MVP 必要）
- Onboarding 基礎收集（暱稱、語言、時段、情緒偏好）
- Daily 心情打卡 + 小任務 + 肯定語
- Diary 文字日記 + 週回顧基礎
- SOS 一鍵求助 + 預設訊息 + 安撫流程
- Profile 個人化設定（主題色、提醒）
- 在地化（繁中/英語切換）

### P1（核心成長）
- 個人化內容推薦（依心情/觸發）
- 微介入工具箱（呼吸、感官回錨）
- 週回顧視覺化強化（趨勢圖）
- 連續使用回饋（徽章/溫柔回饋）
- 內容系統（可更新文案/語句）

### P2（付費成長）
- 付費解鎖機制（訂閱/一次性工具包）
- 進階回顧報告
- 進階陪伴模式（語音/角色）
- 正念導引內容包

### P3（商業與合作）
- 企業/機構版本
- 社群/真人課程入口
- 內容合作與品牌包

## 後續必要步驟（建議路徑）

4. 技術架構確認（本機/雲端/內容配置）

6. 試用與迭代

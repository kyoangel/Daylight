# Daylight 任務分解（Stories/Tasks）與工期估算（P0）

更新日期：2026-02-03

## 估算假設
- 團隊：1 位 Flutter 開發 + 1 位設計（或同一人分擔）
- 工期單位：人日（PD）
- 為避免低估，已含基本測試與調整

---

## EPIC 0：基礎與框架（共 3-5 PD）
1. 導航與路由整理（確保所有頁面可進入）
- Task：底部導覽 + 路由整理
- 估算：0.5-1 PD

2. 全域主題與字體基礎
- Task：色彩、字級、按鈕樣式統一
- 估算：0.5-1 PD

3. 基本在地化框架（繁中/英語）
- Task：i18n 架構、字串資源
- 估算：1-2 PD

---

## EPIC 1：Onboarding（共 4-6 PD）
1. OB-01 歡迎頁
- Task：畫面與 CTA
- 估算：0.5 PD

2. OB-02 基本設定
- Task：暱稱、語言、使用時段
- 估算：1-1.5 PD

3. OB-03 情緒偏好
- Task：多選/單選 UI + 保存
- 估算：1.5-2 PD

4. OB-04 安全提示
- Task：內容頁 + CTA
- 估算：0.5 PD

5. Onboarding 流程狀態管理
- Task：流程狀態保存（暫存）
- 估算：0.5-1 PD

---

## EPIC 2：日常 Daily（共 5-7 PD）
1. DA-01 日常首頁
- Task：心情滑桿、小任務卡、肯定語卡
- 估算：1.5-2 PD

2. DA-02 完成回饋
- Task：完成動畫 + 回饋訊息
- 估算：1 PD

3. DA-03 晚安反思
- Task：引導問題 + 輸入保存
- 估算：1-1.5 PD

4. Daily 資料儲存
- Task：本機保存每日記錄
- 估算：1-1.5 PD

---

## EPIC 3：日記 Diary（共 4-6 PD）
1. DI-01 日記首頁
- Task：心情選擇 + 文字輸入
- 估算：1-1.5 PD

2. 日記模板
- Task：感謝/放下/希望模板
- 估算：0.5-1 PD

3. DI-02 週回顧
- Task：基本趨勢圖 + 文案
- 估算：2-3 PD

---

## EPIC 4：陪伴 Companion（共 4-6 PD）
1. CO-01 模式選擇
- Task：三種模式卡片
- 估算：0.5-1 PD

2. CO-02 對話頁
- Task：聊天 UI + 快捷回覆
- 估算：2-3 PD

3. 陪伴流程狀態
- Task：10 分鐘計時 + 結束回饋
- 估算：1-2 PD

---

## EPIC 5：SOS（共 4-6 PD）
1. SO-01 SOS 首頁
- Task：一鍵求助 + 聯絡人
- 估算：1.5-2 PD

2. SO-02 安撫流程
- Task：呼吸/掃描/回錨流程
- 估算：2-3 PD

3. 區域化危機資源
- Task：台灣資源文案
- 估算：0.5-1 PD

---

## EPIC 6：Profile（共 3-5 PD）
1. 基本個人設定
- Task：暱稱、頭像、主題色
- 估算：1.5-2 PD

2. 提醒時段
- Task：時間選擇 + 保存
- 估算：1-1.5 PD

3. 語言切換
- Task：語言設定與立即生效
- 估算：0.5-1.5 PD

---

## EPIC 7：測試與調整（共 3-5 PD）
1. 基本 UI 測試與調整
- 估算：1-2 PD

2. 核心流程測試（P0）
- 估算：1-2 PD

3. 修正與優化
- 估算：1 PD

---

## P0 粗估總工期
- 合計：30 - 44 人日（約 6 - 9 週，若單人全職）
